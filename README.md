Viene introdotto il concetto di algoritmo e di programma, unitamente ai principali strumenti coinvolti nel ciclo di vita del software. Viene introdotta la programmazione strutturata ed il paradigma imperativo, ponendo l'accento sulla modularizzazione del software.
